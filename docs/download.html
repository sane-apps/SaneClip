Download SaneClip

Use this page when an older SaneClip build could not update itself. The goal is one clean app in /Applications, not two copies with different names.

Important

Do not unzip SaneClip directly inside /Applications. That is how macOS can create SaneClip 2.app next to the old one.

Manual install

Download the latest direct build, open it from Downloads, then drag SaneClip.app onto Applications and choose Replace if Finder asks.

Current direct download link: loading latest release from the live appcast.

1
Download to Downloads

Keep the ZIP in Downloads first. Do not extract it directly into /Applications.

2
Open the ZIP from Downloads

macOS will unpack a fresh SaneClip.app. Leave the unpacked app in Downloads for the next step.

3
Drag SaneClip.app onto Applications

When Finder asks, choose Replace. After that, launch the copy in /Applications.

4
Keep one copy

If you see both SaneClip.app and SaneClip 2.app, delete the extra one and keep only /Applications/SaneClip.app.

If you already ended up with two copies

  • Keep the newest /Applications/SaneClip.app.
  • Delete SaneClip 2.app if Finder created it.
  • Delete any extra ~/Applications/SaneClip.app copy unless you intentionally use a separate dev build.

Need help?

If the download still does not replace the old app cleanly, email hi@saneapps.com or use the support page.
